Networking Concepts Used in the Creation of this Website

  • The domain github.io uses DNS to translate the website name into an IP address.
  • IP provides a unique address to the server that our website is hosted on to allow clients to identify and connect to it.
  • Our website uses the HTTPS protocol, which encrypts the messages it sends and ensures secure communication between client and server.
