Networking Concepts Used in the Creation of this Website

  • The domain github.io uses DNS to translate the website URL into an IP address.
  • IP provides a unique address to the server that our website is hosted on to allow clients to identify and connect to it.
  • Our website uses the HTTPS protocol, which encrypts the messages it sends and ensures secure communication between the client and the server.

Technologies Used to Build this Website

  • Github Pages provides free hosting for our website.
  • HTML was used to build the foundation of our website.
  • CSS was used to help design the visuals and styles of the website.
